Portfolio

Bushyman

Bushyman is a 3D platformer collectathon game where Busyman and his companion Bag Boy try to save the world from Fevil the Evil. They have to collect golden leaves and Rubber Ducks to open doors to new worlds and evil bosses to eventualy stop Fevil from destroying the world.

Dialogue System

The dialogue sytem is made out of multiple scripts. One script is on the npc that has all the information on it like text/sprites and sfx and one script that manages everything. The manager puts all the info from the npc into a dialogue box and displays the text letter by letter.

    • 3rd Person Movement

    The Movement script takes care of all the input from the player. The script lets you walk/run/hover/swim/attack and takes care of the 3rd person camera controls.

    • Weeping Willows

    Weeping Willows is a 2D horror platformer game about a little girl called Mya. Every time you reach the end of the level the world turns dark and you start from the beginning all over again but every time this happens it gets more and more dark and spooky around you.

    2D Movement

    The 2D movement script takes care of the player inputs. It allows the player to walk around and jump. This script was made so it could be played on keyboard but also on a Arcade Machine

    • Player Health

    The player health script keeps track of the amount of health the player has and updates the hearts in the UI. It also updates it once the player takes damage

    • Banananoid

    Banananoid is a 3D game inspired by Arcanoid. In Banananoid you play as a banana tree that has to run away from hungry monkeys. You have to shoot bananas at the monkeys and bounce them between the monkeys to kill them and stay alive as long as possible.

    Monkey Spawner

    The spawner script randomly picks one spawn location out of a list of possible spots and spawns a monkey there. The script also chooses what type of the 3 monkeys to spawn in

    • Points Multiplier

    The points multiplier script has a internal multiplier that multiplies the amount of points the player gets from hitting a monkey. The multiplier will reset back to 0 when the internal timer runs out but can be reset by hitting another monkey within this time frame.
